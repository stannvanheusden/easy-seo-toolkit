== Description ==

**WordPress SEO plugin for modern websites**
Easy SEO Toolkit is a WordPress SEO plugin that integrates seamlessly with Gutenberg, Classic Editor and the Elementor Pagebuilder. 

The plugin enables the possibility to add keywords, meta titles and descriptions. Next to that it generates an XML Sitemap for your site.

**Elementor SEO plugin**
Stop switching back from your Elementor pages to the dashboard to configure your SEO settings. This plugins adds the functionality to Elementor to:

* Add Keywords
* Add Meta Titles
* Add Meta Descriptions
* Add Twitter Cards automaticly
* Add XML Sitemap to your site!

All from the "Page Settings" section.
